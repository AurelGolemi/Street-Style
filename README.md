# Street Style

A modern e-commerce inspired clothing brand landing page built with Next.js and TypeScript, focused on performance, responsive design, and reusable UI architecture.

---

## 🚀 Overview

Street Style is a responsive web application designed to simulate a modern clothing brand storefront. The goal of this project was to practice scalable component architecture, clean UI design, and production-level deployment workflows using modern frontend technologies.

---

## 🛠 Tech Stack

Frontend:
- Next.js
- React
- TypeScript
- Tailwind CSS

Deployment:
- Vercel

Tooling:
- Git
- ESLint
- Prettier

---

## ✨ Features

- Fully responsive layout (mobile-first)
- Reusable product card components
- Optimized images
- Clean grid system
- SEO metadata configuration
- Structured component hierarchy
- Performance optimization using Next.js features

---

## 🧠 Architecture Highlights

- Component-based architecture for scalability
- Clear separation between UI components and layout logic
- Organized folder structure for maintainability
- Type-safe props and state management using TypeScript
- Production-style deployment workflow via Vercel

---
